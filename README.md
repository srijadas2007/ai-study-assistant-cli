# AI Study Assistant CLI

## Introduction

This project is a Command Line Interface (CLI) based application developed using Python. It helps users learn basic concepts of Artificial Intelligence and Machine Learning in an interactive way.

---

## Features

* Ask questions related to AI and ML
* Take quizzes to test knowledge
* Identify weak topics based on incorrect answers
* Explore related topics using Breadth First Search (BFS)
* Get recommended topics based on performance

---

## Technologies Used

* Python
* Graph (for topic connections)
* Queue (for BFS traversal)

---

## How to Run the Project

### Step 1: Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/ai-study-assistant-cli
cd ai-study-assistant-cli
```

### Step 2: Run the program

```bash
python main.py
```

---

## How to Use

After running the program, you will see a menu:

1. Ask Question → Enter any AI/ML related question
2. Take Quiz → Answer questions and get score
3. Weak Topics → View topics you answered incorrectly
4. Explore Topics → Enter a topic to see related topics
5. Recommended Topics → View suggested topics based on performance
6. Exit → Close the application
